# Portfolio Website – Interactive Media Project

Dieses Projekt ist eine persönliche Portfolio-Webseite, die im Rahmen des Moduls **Interaktive Medien 5 (IM5)** entstanden ist. Die Webseite dient als zentrale Plattform zur Präsentation von Arbeiten, visuellen Inhalten und Projekten aus dem Bereich Multimedia / Interaktive Medien.

Der Fokus liegt auf einer klaren Gestaltung, einer durchdachten User Experience sowie einer sauberen technischen Umsetzung im Web.

---

## Projektidee & Fragestellung

Ziel des Projekts ist es, ein persönliches Portfolio zu entwickeln, das nicht nur Arbeiten zeigt, sondern auch eine eigene visuelle Sprache und Haltung vermittelt.

Zentrale Fragestellung:

**Wie kann eine Portfolio-Webseite gestaltet werden, die Persönlichkeit, Inhalte und visuelle Qualität gleichwertig vermittelt und gleichzeitig technisch sauber umgesetzt ist?**

---

## Umsetzung

- **Projektform:** Webprojekt  
- **Arbeitsform:** Einzelarbeit  
- **Technologien:**
  - HTML
  - CSS
  - JavaScript

- **Schwerpunkte:**
  - Strukturierte Informationsarchitektur
  - Visuelles Storytelling
  - Interaktive Elemente (Hover-Effekte, Animationen, Grid-Layouts)

---

## Setup / Nutzung

Die Webseite ist als statische Website umgesetzt.

### Lokale Nutzung
- Projektordner öffnen
- `index.html` im Browser öffnen

### Online-Version
- URL zur Portfolio-Webseite (falls online gehostet)

**Hinweis zur Darstellung:**  
Die aktuelle Version der Webseite ist primär für die Nutzung auf **Laptops und Desktop-Bildschirmen** konzipiert. Eine optimierte Darstellung für mobile Endgeräte ist bewusst nicht Teil dieses Projektumfangs und wird in einem möglichen nächsten Entwicklungsschritt umgesetzt.

---

## Ergebnis

Entstanden ist eine funktionale und visuell reduzierte Portfolio-Webseite, die:
- Arbeiten klar und strukturiert präsentiert
- visuelle Inhalte in den Vordergrund stellt
- eine konsistente gestalterische Sprache verfolgt
- technisch sauber umgesetzt ist und einfach erweitert werden kann

---

## Learnings

- Konzeption und Umsetzung eines eigenständigen Webprojekts
- Vertiefung von HTML, CSS und JavaScript
- Gestaltung von Layouts mit Fokus auf visuelle Wirkung
- Verbindung von Gestaltung, Technik und User Experience
- Selbstorganisation und Projektplanung über einen längeren Zeitraum

---

## Herausforderungen

- Definition eines klaren Projektumfangs
- Balance zwischen gestalterischer Freiheit und Nutzerführung
- Entscheidung, welche Inhalte gezeigt und welche bewusst weggelassen werden
- Gestaltung und Umsetzung für unterschiedliche Bildschirmgrössen mit Fokus auf Desktop-Nutzung
